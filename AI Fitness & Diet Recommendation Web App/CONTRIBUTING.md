# Contributing to AI Fitness & Diet Web App

First off, thank you for considering contributing to our project!

## How to Contribute
1. **Fork** the repository on GitHub.
2. **Clone** your fork locally and create a new branch (`git checkout -b feature/your-feature-name`).
3. **Commit** your changes with clear, descriptive commit messages.
4. **Push** your branch to your fork (`git push origin feature/your-feature-name`).
5. **Open a Pull Request** against the `main` branch of this repository.

## Issues
Before starting major work, please open an Issue to discuss it first. Bug reports and feature requests are welcome!
